# MIP PVL Assistant v12.6.0

Versão redesenhada conforme a arquitetura funcional definida:

- Operações
- Upload do MIP
- SADIPEM
- Pesquisa PVL/API

## Objetivos
- Informar documentos, canal, fluxo e exigências por tipo de operação e data.
- Comparar novo MIP com a base normativa atual do sistema.
- Traduzir exigências para as abas/campos do SADIPEM.
- Pesquisar PVL na API pública e avaliar o pleito conforme regras do MIP.

## Execução
```bash
pip install -r requirements.txt
streamlit run app.py
```
