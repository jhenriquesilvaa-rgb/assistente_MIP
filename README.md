# MIP PVL Assistant v12.5.1

Versão saneada para publicação manual no GitHub e deploy no Streamlit.

## Arquivos principais

- `app.py`: interface principal do app.
- `decision_engine_v10.py`: pesquisa de PVLs, integração multifonte e diagnóstico técnico.
- `rules_operacoes_v10.py`: catálogo base de checklist por modalidade.
- `requirements.txt`: dependências mínimas.

## O que esta versão entrega

- Busca de PVLs por número do PVL, processo, ente, UF, ano, status e tipo de operação.
- Seleção de um resultado para diagnóstico detalhado.
- Integração com endpoints públicos complementares do SADIPEM.
- Diagnóstico por família da operação, status, data e sinais multifonte.
- Estrutura limpa para copiar e colar arquivo por arquivo no GitHub.
