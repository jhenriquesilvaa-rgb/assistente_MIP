# MIP PVL Assistant v12.6.1

Versão completa com o passo seguinte já incorporado.

## Melhorias sobre a v12.6.0
- Corrige a importação inválida em startup.
- Endurece a leitura da API pública com aliases de campos.
- Melhora a pesquisa de PVL com tentativas por `eq` e `ilike`.
- Lista múltiplos PVLs encontrados antes da avaliação.
- Mantém a arquitetura funcional em quatro abas.

## Execução
```bash
pip install -r requirements.txt
streamlit run app.py
```
